# 🍽️ Restaurant Bill Generator

A simple and interactive **Restaurant Bill Generator** built using **HTML, CSS, and JavaScript**. Users can select the quantity of menu items, and the application automatically calculates the **Subtotal**, **GST (5%)**, and **Final Total Bill**.

---

## 📌 Features

- 🍕 Display restaurant menu with prices
- 🔢 Select quantity for each item
- 🧮 Automatic bill calculation
- 💰 Calculates 5% GST
- 📄 Displays detailed bill summary
- 📱 Responsive and user-friendly interface

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
4. Enter the quantity for the required food items.
5. Click **Generate Bill**.
6. View the generated bill with GST included.

---

## 🧾 GST Calculation

```
GST = Subtotal × 5%

Total Bill = Subtotal + GST
```

---

## 📸 Sample Output

```
Pizza × 2        ₹500

-------------------------
```
